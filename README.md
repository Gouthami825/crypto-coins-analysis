#  Crypto Coins Data Analysis

##  Project Overview

This project focuses on analyzing cryptocurrency dataset using Python.

The notebook performs:

- Data loading
- Data inspection
- Data cleaning
- Handling missing values
- Basic statistical analysis
- Exploring cryptocurrency metrics

The objective of this project is to understand cryptocurrency market data and extract meaningful insights from structured financial information.

---

##  Dataset Information

The dataset contains cryptocurrency-related details such as:

- Coin Name
- Symbol
- Price
- Market Capitalization
- Volume
- Other financial attributes

Each row represents one cryptocurrency and its corresponding market data.

---

##  Technologies & Libraries Used

- Python  
- NumPy  
- Pandas  

---

##  Project Workflow

1. Import required libraries  
2. Load the dataset  
3. Explore dataset structure  
4. Check for missing values  
5. Clean and preprocess data  
6. Perform statistical analysis  
7. Draw conclusions from observations
